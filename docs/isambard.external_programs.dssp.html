isambard.external_programs.dssp module¶

This module provides an interface to the program DSSP.

For more information on DSSP see [1].

References

[1]Kabsch W, Sander C (1983) “Dictionary of protein secondary structure: pattern recognition of hydrogen-bonded and geometrical features”, Biopolymers, 22, 2577-637.

extract_helices_dssp(in_pdb)[source]¶

Uses DSSP to find alpha-helices and extracts helices from a pdb file.

Returns a length 3 list with a helix id, the chain id and a dict containing the coordinates of each residues CA.

Parameters:in_pdb (string) – Path to a PDB file.

find_ss_regions(dssp_residues)[source]¶

Separates parsed DSSP data into groups of secondary structure.

Notes

Example: all residues in a single helix/loop/strand will be gathered into a list, then the next secondary structure element will be gathered into a separate list, and so on.

Parameters:dssp_residues ([list]) –
Each internal list contains:
[0] int Residue number [1] str Secondary structure type [2] str Chain identifier [3] str Residue type [4] float Phi torsion angle [5] float Psi torsion angle
Returns:fragments – Lists grouped in continuous regions of secondary structure. Innermost list has the same format as above.
Return type:[[list]]
